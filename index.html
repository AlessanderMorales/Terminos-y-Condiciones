OpenHands – Términos y Condiciones & Política de Privacidad

Última actualización: 2025

Términos y Condiciones

1. Aceptación de los Términos

Al descargar o utilizar la aplicación OpenHands, el usuario acepta estos Términos y Condiciones. Si el usuario no está de acuerdo, debe dejar de usar la aplicación.

2. Descripción del Servicio

OpenHands es una aplicación gratuita diseñada para apoyar la comunicación con personas sordomudas.

Sus funciones incluyen:

  • Interpretar señas usando la cámara y convertirlas en letras.
  • Mostrar mediante un avatar cómo realizar una seña cuando el usuario escribe una letra.

3. Registro y Cuenta de Usuario

Para usar ciertas funciones, el usuario puede crear una cuenta proporcionando nombre, correo electrónico y contraseña.

Es la única información personal que se recopila.

4. Licencia de Uso

El usuario recibe una licencia limitada, personal y no transferible para usar la app. No está permitido copiar, vender, modificar ni desensamblar el software.

5. Uso de la Cámara

La app requiere acceso a la cámara para interpretar señas. OpenHands no almacena imágenes ni videos.

6. Limitación de Responsabilidad

No se garantiza que la interpretación de señas sea 100% exacta ni que el servicio funcione sin errores. El usuario utiliza la app bajo su propia responsabilidad.

7. Propiedad Intelectual

Todo el contenido, diseño, funciones, avatar y software pertenecen al desarrollador de OpenHands.

8. Modificaciones

El desarrollador puede actualizar estos términos cuando sea necesario.

9. Ley Aplicable

Estos términos se rigen por las leyes del Estado Plurinacional de Bolivia.

Política de Privacidad

1. Información que recopilamos

OpenHands únicamente recopila la siguiente información para registro de usuarios:

  • Nombre
  • Correo electrónico
  • Contraseña

No se recopilan ni almacenan imágenes, videos o datos biométricos.

2. Uso de la Información

La información de registro se utiliza exclusivamente para:

  • Crear la cuenta del usuario
  • Permitir el inicio de sesión
  • Mantener acceso seguro a sus datos

No se comparte información con terceros.

3. Uso de la Cámara

La cámara se usa solo para interpretar señas en tiempo real. No se guarda ni envía ningún contenido visual.

4. Seguridad de los Datos

La información del usuario se almacena de manera segura. Sin embargo, ningún sistema es 100% infalible, por lo que no se puede garantizar seguridad absoluta.

5. Derechos del Usuario

El usuario puede:

  • Solicitar eliminación de su cuenta
  • Pedir actualización de sus datos
  • Dejar de usar la app en cualquier momento

6. Cambios en la Política

Esta Política de Privacidad puede actualizarse ocasionalmente. Se notificará a través de la app si se realizan cambios importantes.
