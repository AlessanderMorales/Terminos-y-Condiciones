Términos y Condiciones

1. Aceptación de los Términos

Al descargar o utilizar la aplicación OpenHands, el usuario acepta estos Términos y Condiciones. Si el usuario no está de acuerdo, debe dejar de usar la aplicación.

2. Descripción del Servicio

OpenHands es una aplicación gratuita diseñada para apoyar la comunicación mediante la Lengua de Señas Boliviana (LSB). Sus funciones incluyen:

  • Interpretar señas utilizando la cámara y convertirlas en texto.
  • Generar una animación de un avatar mostrando la seña correspondiente a la letra ingresada.

3. Registro y Cuenta de Usuario

Para acceder a algunas funciones, se requiere un registro con nombre, correo electrónico y contraseña. Esta es la única información personal recopilada.

4. Licencia de Uso

OpenHands otorga una licencia limitada, personal y no transferible para usar la aplicación. Está prohibido modificar, vender o desensamblar el software.

5. Uso de la Cámara

La cámara se utiliza únicamente cuando el usuario activa el modo de detección de señas. OpenHands no almacena fotografías, videos ni datos biométricos.

6. Limitación de Responsabilidad

La aplicación puede no ser 100% precisa debido a iluminación, posición de las manos u otros factores. No sustituye a intérpretes profesionales.

7. Propiedad Intelectual

Todo diseño, código, funciones y contenido pertenecen al desarrollador de OpenHands.

8. Modificaciones

Podemos actualizar estos términos en cualquier momento. Se publicarán en esta página.

9. Ley Aplicable

Estos términos se rigen por las leyes del Estado Plurinacional de Bolivia.

Política de Privacidad

OpenHands ha desarrollado la aplicación como una herramienta gratuita para apoyar la comunicación mediante Lengua de Señas Boliviana (LSB). Este servicio se proporciona sin costo alguno y tal cual está disponible.

Esta página informa sobre nuestras políticas respecto a la recopilación, uso y divulgación de Información Personal.

1. Uso de la Cámara y Procesamiento de Imágenes

Nuestra aplicación requiere acceso a la cámara para la función principal de “Detección de Señas en Tiempo Real”.

  • Recopilación: La aplicación captura cuadros de video únicamente cuando el usuario activa el modo cámara.
  • Transmisión y Procesamiento: Las imágenes se envían de forma segura a nuestro servidor privado para ser analizadas mediante algoritmos de IA (MediaPipe).
  • No Almacenamiento: Las imágenes NO se guardan. Se procesan temporalmente en RAM y luego se eliminan permanentemente. No generamos bases de datos biométricas.

2. Funcionalidad de Texto a Seña (Servicios de Terceros)

La función de “Texto a Seña” utiliza un avatar animado proporcionado por el servicio externo sign.mt.

  • Datos Compartidos: Su dirección IP y la solicitud de texto pueden ser procesadas por sign.mt.
  • Política de Terceros: Esta función está sujeta a las prácticas de privacidad de sign.mt.

3. Historial de Traducciones (Almacenamiento Local)

La aplicación guarda un historial de traducciones en su dispositivo:

  • Los datos son locales.
  • No se suben a la nube.
  • Se borran al desinstalar la app.

4. Permisos Solicitados

  • CAMERA: Necesario para detectar señas.
  • INTERNET: Requerido para conectar con nuestros servidores y con el avatar de sign.mt.

5. Seguridad de los Datos

Utilizamos comunicación cifrada (WebSockets/HTTPS). Aunque se siguen prácticas seguras, ningún sistema puede garantizar protección absoluta al 100%.

6. Descargo de Responsabilidad (Precisión)

La tecnología puede cometer errores. No debe usarse en situaciones críticas como médicas o legales.

7. Privacidad de los Niños

No recopilamos información de menores de 13 años de forma intencional.

8. Cambios en esta Política

Podemos actualizar esta política. Se recomienda revisarla periódicamente.
